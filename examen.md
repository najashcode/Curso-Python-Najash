# Operaciones con listas y bucles:

* Crea una lista llamada inventario y añade el diccionario producto a esta lista.
* Solicita al usuario que ingrese tres productos más, utilizando el mismo proceso.
* Utiliza un bucle para añadir cada producto a la lista inventario.

# Condicionales y operadores de comparación:

Crea una función llamada consultar_precio que pida al usuario el nombre de un producto y, si está en el inventario, imprima su precio.
Si el producto no está en el inventario, muestra un mensaje indicando que no se encontró.

# Operadores matemáticos y acumulación:

Calcula el valor total del inventario usando un bucle para sumar el valor de cada producto (precio * cantidad) y muestra el resultado.

# Uso de tuplas y conjuntos:

Extrae y almacena todos los nombres de los productos en una tupla llamada productos_disponibles.
Utiliza un conjunto para almacenar los precios únicos de los productos en el inventario.

# Salida de datos (output):

Usa print() y f-strings para mostrar:
El inventario completo (nombre, precio, cantidad, disponibilidad).
El valor total del inventario.
La lista de productos disponibles (tupla).
El conjunto de precios únicos.
